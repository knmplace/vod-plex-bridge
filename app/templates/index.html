What Gets Refreshed
1. Dump RegenRegenerates stream_mapping.json from Dispatcharr's DB (picks up stream_id changes from M3U refresh) 2. Catalog RefreshPages through all Dispatcharr VOD movies — adds new movies, updates metadata, detects UUID changes and clears stale caches 3. Stream MappingApplies fresh stream_ids to all movies in the bridge DB 4. Activated RefreshFor activated movies: if stream_id changed, clears old head/tail cache and re-fetches from provider (sequential, one at a time) 5. Dead ScanRemoves movies no longer in Dispatcharr's catalog — deactivates, removes STRM files, removes from Plex
